# Bigtreetech WiKi NEO

### install requirement 

#### ArchLinux

``` shell
sudo pacman -S yarn
```

#### Debian

``` shell
sudo apt install yarn
```

#### Windows

First, download and install Node.js

Then in PowerShell use following command install yarn

``` shell
npm install --global yarn
```

### Install Package 

``` shell
cd ~/wiki-dev
yarn install 
```

### Run wiki

``` shell
yarn run start
```

### Build wiki

``` shell
yarn run build
```
