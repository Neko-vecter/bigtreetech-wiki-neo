## 产品简介

EZ 5160 Pro 使用 TMC 5160 步进电机控制芯片，外置 MOSFET，最大电压可以达到 48V。

## Specification

| EZ 5160 Pro | Specification      |
| ----------- | ------------------ |
| 外观尺寸    | $18.5 mm * 20 mm$  |
| 驱动芯片    | TMC5160-TA         |
| 输入电压    | 8V-48V             |
| 输出电流    | 2.5A RMS 3.0A Peak |
| 最大细分    | 256                |
| 工作模式    | SPI                |
