## 产品简介

TMC 5160 T Plus 是一个外置步进电机驱动。使用大功率 MOSFET 最大电压为 56V。

## Specifications

| TMC 5160 T Plus                | 参数                |
| ------------------------------ | ------------------- |
| TMC5160T Plus 包括散热器的尺寸 | 64 x 56 x 32.55mm   |
| TMC5160T Plus 板子尺寸         | 58 x 50 x 28mm      |
| 驱动芯片                       | TMC5160-TA          |
| 输入电压                       | 8V - 56V            |
| 驱动电流                       | 10.6A RMS, 15A Peak |
| 板载电容                       | 2 x $560uF$         |
| 驱动模式                       | SPI, Step/DIR       |
| 采样电阻                       | $22 m\Omega$        |
