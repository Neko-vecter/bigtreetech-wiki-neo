## **功能亮点**

- 整体体积小，重量轻；
- 具备温度补偿功能；
- 高效率调平；
- 应用广泛，兼容性强；
- 精度高，稳定性强；
- 非接触式；

## **规格**

| **产品名称** | **Eddy**                                                     |
| ------------ | ------------------------------------------------------------ |
| **产品重量** | **6g**                                                       |
| **电压**     | **5V**                                                       |
| **静态电流** | **30mA**                                                     |
| **工作电流** | **30mA**                                                     |
| **接线**     | **Eddy, Eddy Duo：4pin, 间距1.5mm** |
